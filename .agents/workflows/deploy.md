# GitHub Deployment Workflow

// turbo-all
1.  Open the terminal in `c:\Users\HP\Downloads\Kesav portfolio`.
2.  Initialize git:
    ```bash
    git init
    ```
3.  Add all files:
    ```bash
    git add .
    ```
4.  Commit changes:
    ```bash
    git commit -m "Initialize portfolio"
    ```
5.  Link your remote:
    ```bash
    git remote add origin https://github.com/kesava895/kesava-portfolio.git
    ```
6.  Push to main:
    ```bash
    git push -u origin main
    ```

> [!IMPORTANT]
> Change the URL in Step 5 to your actual repository URL if it's different.
